# Simple Calculator

A minimal calculator built with HTML, CSS, and JavaScript. Clean orange and black design with circular buttons and expression display.

## Features

- **Basic operations** — Add (+), subtract (−), multiply (×), divide (÷)
- **Expression display** — Shows the full expression (e.g. `1 + 2`) above the result when you press =
- **Clear (C)** — Reset the calculator
- **Negate (±)** — Toggle positive/negative
- **Percent (%)** — Divide current value by 100
- **Decimal input** — Supports numbers like 3.14

## How to Run

1. Open the project folder.
2. Double-click `index.html` to open it in your default browser, or right-click → **Open with** and choose a browser.

No build step or server required.

## Tech Stack

- **HTML** — Structure and buttons
- **CSS** — Centered layout, grid, orange/black styling, circular buttons
- **JavaScript** — Input handling, arithmetic, expression and result display

## Design

- Calculator centered on the page
- Circular number/operator buttons
- Orange (#ff8c00) for operators and equals, black/dark gray for background and number keys
- Expression line in orange above the main result display
